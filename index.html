Шаг 1.
Находим в верхнем левом углу зеленую кнопку


Шаг 2.
На открывшейся странице в поле repository name вводим имя репозитория


Шаг 3.
Скролим вниз и жмем на кнопку Create repository


Шаг 4.
Ожидаем несколько секунд и на открывшейся странице клик по надписи uploading an existing file


Шаг 5.
Откроется drag-drop поле клик по надписи


Шаг 6.
Откроется диалоговое окно, выбираем папку с проектом ,открываем , выделяем и перетаскиваем все файлы


Шаг 7.
После окончания загрузки закрываем дталоговое окно, скрол вниз и клик по кнопке


Шаг 8.
После того как Гит прожует то, чем вы его накормили вы увидите такую страницу, здесь нас интересует категория SETTING


Шаг 9.
Скролим вниз и ищем раздел GitHub Pages, клик по селекту


Шаг 10.
Выбираем main и жмем save


Шаг 11.
Вас перебросит вверх страницы, скрол вниз к разделу GitHub Pages и если вы наблюдаете подобную ссылку с вашим логином и названием проекта, значит все прошло успешно Можете кликнуть на ссылку, но сперва вы увидите ошибку 404, это потому, что ваш проект проходит процесс инициализации. Обычное время ожидания 5-7 минут. По истечении которых вы можете увидеть свое творение в сети
